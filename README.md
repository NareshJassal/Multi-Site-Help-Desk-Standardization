# Multi-Site Help Desk Standardization Project
**Project Lead | 2021-2023 | Lidl US**

## Overview
Consolidated IT help desk operations across 15 retail locations, creating standardized ticketing procedures, tiered support model (L1/L2), and comprehensive training program.

## Challenge
- Inconsistent SLA compliance across locations (12% miss rate)
- No standardized escalation procedures
- Average response time 20% above industry benchmark
- 52% training/certification completion rate
- Support bottlenecks during peak retail hours

## Solution

### Tiered Support Model
- Designed L1/L2 support structure with clear escalation criteria
- Created skill-based routing in ServiceNow
- Implemented real-time queue monitoring and load balancing

### Process Standardization
- Consolidated 15 different ticketing workflows into one unified system
- Standardized incident categorization and priority levels
- Developed shift handover procedures for 24/7 coverage

### Training & Development
- Built structured learning paths for Windows, Mac OS, networking basics
- Created 12 technical troubleshooting guides
- Delivered monthly hands-on training sessions for 60+ staff

## Technologies
- **ITSM:** ServiceNow
- **Systems:** Windows 10/11, Mac OS, Active Directory
- **Network:** TCP/IP, DNS, DHCP, VPN
- **Tools:** Remote Desktop, Citrix, SCCM

## Results

| Metric | Before | After | Improvement |
|--------|--------|-------|-------------|
| Average Response Time | Baseline | -15% | 15% faster |
| SLA Miss Rate | 12% | 5% | -58% |
| SLA Compliance | ~88% | ~94% | +6% |
| Training Completion | 52% | 84% | +32% |
| Support Bottlenecks | High | -28% | 28% reduction |
| Repeat Tickets | Baseline | -26% | 26% fewer |
